✈ TravelAway – Full Stack Travel Booking System
📌 Overview

TravelAway is a full-stack travel booking web application developed using ASP.NET Core Web API and Angular.
The system follows a layered 3-tier architecture ensuring separation of concerns and scalability.

🏗 Architecture

Presentation (Angular)
        ↓
Web API (ASP.NET Core)
        ↓
Business Logic Layer
        ↓
Data Access Layer
        ↓
SQL Server

🚀 Tech Stack

ASP.NET Core Web API
Angular
C#
SQL Server
RESTful APIs
3-Tier Architecture
Entity Framework / ADO.NET (whichever you used)

🔥 Key Features

User Registration & Login
Travel Package Management
Booking Management
Admin Dashboard
CRUD Operations
API-based Communication
Structured Exception Handling

🛠 How to Run

Backend
Update connection string in appsettings.json
Run dotnet run
Frontend
npm install
ng serve

🎯 Learning Outcomes

Designed layered architecture
Built RESTful APIs
Implemented client-server communication
Applied SOLID principles
